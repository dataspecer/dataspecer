# Specification Package Documentation

This directory contains documentation for the `@dataspecer/specification` package.

## Contents

- [generator-configuration-storage.md](./generator-configuration-storage.md) - Comprehensive guide on where and how generator configurations are stored in the system, including semantic-level and data specification-level storage.

## Overview

The specification package provides functionality for:

- Data specification modeling and management
- Generator configuration storage and merging
- Semantic model aggregation and composition
- Artifact configuration and generation

For more details on specific topics, see the individual documentation files.
